# envi
```
envi streamlines the management and synchronization of .env files among different environments or team members, facilitating consistent configuration across development workflows.

Usage:
  envi [command] [flags]
  envi [command]

Available Commands:
  auth        Display commands for authenticating envi with an account
  completion  Generate the autocompletion script for the specified shell
  help        Help about any command
  pull        Pulls the latest .env file from the remote server and replaces the local .env file with it.

Flags:
  -h, --help      help for envi
  -v, --version   version for envi

Use "envi [command] --help" for more information about a command.
```
